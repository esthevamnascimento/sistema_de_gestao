📦 Como instalar no Servidor da Empresa agora?
Quando o comando terminar, você vai notar que o PyInstaller criou algumas pastas novas no seu projeto.
Abra a pasta chamada dist (de distribution). Lá dentro estará o seu reluzente SistemaGestaoCRZ.exe.

Para o sistema funcionar perfeitamente no servidor da empresa, você só precisa copiar para lá uma pasta contendo 3 coisas:

1.O seu novo SistemaGestaoCRZ.exe.

2.A sua pasta templates (onde estão os seus HTMLs).

3.O seu arquivo sistema.db (se quiser levar os dados que já cadastrou, senão o sistema cria um novo vazio e você roda o .exe de criar usuários).

Pronto! Dê dois cliques no .exe no servidor, autorize o Firewall do Windows (se ele pedir na primeira vez) e o sistema já estará no ar para a empresa inteira pelo IP da máquina!
